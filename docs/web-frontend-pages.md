
A. Front Page
Purpose:
Present the platform, its value, and key features immediately.

Content:
• Short explanation of the problem the platform solves (model discovery, pricing, comparisons)
• Highlight main features: Model Catalog, Pricing Engine, Capability Matching, Benchmarks, Equivalency Suggestions
• Clear CTAs for exploring the catalog or signing up
• Quick preview of top models, cheapest alternatives, and trending benchmarks

B. About Page
Purpose:
Explain how the platform works and where the data comes from.

Content:
• How model data is gathered and updated
• How pricing normalization and cost comparisons work
• How capability tags and benchmark scores are mapped
• How equivalency suggestions are generated
• Links to contribute: reporting model changes, suggesting new providers, API usage

C. Contact Page
Purpose:
Provide communication channels for users, partners, and enterprises.

Content:
• Business email
• Support email
• Partnership / enterprise inquiry form
• Optional: link to GitHub or contribution pages

D. Pricing Page
Purpose:
Let users quickly compare Free, Pro, and Enterprise tiers.

Content:
• Free: basic search, essential filters, limited comparisons
• Pro: unlimited comparisons, advanced filters, historical pricing, cost-optimizer, bulk export, templates, API access
• Enterprise: on-premise sync, internal system integrations, custom benchmarks, admin dashboard

E. Model Catalog Page
Purpose:
Central hub for browsing all AI models.

Content:
• Search + filters (provider, field, capabilities, price range, benchmarks)
• Model cards with core attributes
• Sorting: performance, cost, release recency, popularity

F. Model Details Page
Purpose:
Show deep, structured model information.

Content:
• Provider, release date, capabilities, pricing, modalities, formats
• Benchmarks
• Similar models + cheaper alternatives
• API usage example

G. Comparison Page
Purpose:
Side-by-side model comparison.

Content:
• Pricing normalized per 1M tokens
• Performance charts
• Capability breakdown
• Suggestions for “better value” or “higher performance” options
